# 取得支援
{: #ibmblockchain_support}

***[此頁面有幫助嗎？請告訴我們。](https://www.surveygizmo.com/s3/4501493/IBM-Blockchain-Documentation)***

{{site.data.keyword.blockchainfull}} Platform (IBP) 提供數個用於問題疑難排解及取得支援的通路，這些都與您使用的 IBP 供應項目相關聯。
{:shortdesc}

對於所有 IBP 供應項目，建議先使用[免費數位支援資源](#resources)進行問題疑難排解，並從 {{site.data.keyword.IBM_notm}} 及「Hyperledger Fabric 社群」取得協助。如果您使用「入門範本方案」或「企業方案」網路，則可以導覽至「網路監視器」中的「支援」畫面，其中包含這些資源的鏈結。

如果任一[免費數位支援資源](#resources)都無法解決您的問題，請考量使用下列方式來取得您所使用供應項目的支援。
- **{{site.data.keyword.cloud_notm}} 上的入門範本方案及企業方案**
  將支援案例提交至 {{site.data.keyword.IBM_notm}}。如需相關資訊，請參閱[提交支援案例](#support-cases)。
- **IBP for {{site.data.keyword.cloud_notm}} Private (ICP)**
  如果您已購買 IBP for ICP 授權，而且想要聯絡客戶支援中心，請參閱[存取 IBM Support Community 並開立支援問題單 ![外部鏈結圖示](images/external_link.svg "外部鏈結圖示")](https://www.ibm.com/support/docview.wss?uid=ibm10740041 "{{site.data.keyword.blockchainfull_notm}} Platform for ICP 支援")。
- **{{site.data.keyword.blockchainfull_notm}} Platform Community Edition**
  {{site.data.keyword.blockchainfull_notm}} Platform Community Edition 主要用於探索、開發及測試。{{site.data.keyword.IBM_notm}} 未提供 {{site.data.keyword.blockchainfull_notm}} Platform Community Edition 的支援。

  若為與 ICP 相關的問題，您可以利用 [ICP 的免費數位支援及付費支援 ![外部鏈結圖示](images/external_link.svg "外部鏈結圖示")](https://www.ibm.com/developerworks/community/blogs/fe25b4ef-ea6a-4d86-a629-6f87ccf4649e/entry/Learn_more_about_IBM_Cloud_Private_Support?lang=en_us "IBM Cloud Private 支援")。
- **{{site.data.keyword.blockchainfull_notm}} Platform for AWS**
  {{site.data.keyword.IBM_notm}} 未提供此供應項目的支援，而且不會回應在 {{site.data.keyword.cloud_notm}} 中開啟且與 IBP for AWS 相關的支援案例。

  若為與 AWS 相關的問題，您可以使用[社群支援討論區 ![外部鏈結圖示](images/external_link.svg "外部鏈結圖示")](https://forums.aws.amazon.com/index.jspa "AWS 社群支援討論區") 及 [AWS 超值支援 ![外部鏈結圖示](images/external_link.svg "外部鏈結圖示")](https://aws.amazon.com/premiumsupport/ "AWS 超值支援")。

## 資源及支援討論區
{: #resources}

**{{site.data.keyword.blockchainfull_notm}} 服務文件**
[{{site.data.keyword.blockchainfull_notm}} 服務文件](index.html)（即此文件網站）提供如何開始使用 IBP 的指引。您可以從左導覽器中尋找對應的主題，或是使用頂端的搜尋功能來搜尋任何術語。

**{{site.data.keyword.IBM_notm}} dWAnswers**
[{{site.data.keyword.IBM_notm}}dWAnswers ![外部鏈結圖示](images/external_link.svg "外部鏈結圖示")](https://developer.ibm.com/answers/smartspace/blockchain/ "Blockchain 空間中的問題及回答") 是 IBP 及 Hyperledger Fabric 使用者的社群討論區，其由來自 {{site.data.keyword.IBM_notm}} 的專家所監視。您可以搜尋先前所張貼問題的回答，或提交新的問題。請務必將 **blockchain** 關鍵字併入新問題中。

**範例應用程式**
**Blockchain 範例應用程式**下的[範例應用程式 ![外部鏈結圖示](images/external_link.svg "外部鏈結圖示")](https://github.com/ibm-blockchain "IBM Blockchain 範例應用程式") 提供指引及範例程式碼 Snippet，以協助開發應用程式。

**Hyperledger Fabric 資源**
[Hyperledger Fabric 文件 ![外部鏈結圖示](images/external_link.svg "外部鏈結圖示")](https://hyperledger-fabric.readthedocs.io/en/latest/ "Hyperledger Fabric")、[Fabric 社群 Wiki ![外部鏈結圖示](images/external_link.svg "外部鏈結圖示")](https://wiki.hyperledger.org/projects/fabric "Fabric 社群 Wiki") 及 [Fabric Jira 儀表板 ![外部鏈結圖示](images/external_link.svg "外部鏈結圖示")](https://jira.hyperledger.org/secure/Dashboard.jspa?selectPageId=10104 "Fabric Jira 儀表板") 提供更多有關 Fabric 堆疊的詳細資料。

您也可以在 [Rocket.Chat ![外部鏈結圖示](images/external_link.svg "外部鏈結圖示")](https://chat.hyperledger.org/channel/fabric "Fabric Rocket.Chat 頻道") 中與 Fabric 專家交談，以取得 Fabric 問題的回答。

**Hyperledger Composer 資源**
因為 {{site.data.keyword.IBM_notm}} 未提供使用 Hyperledger Composer 進行正式作業實作的支援，所以鼓勵您在使用展示及概念驗證時參與 [Hyperledger Composer 社群 ![外部鏈結圖示](images/external_link.svg "外部鏈結圖示")](https://chat.hyperledger.org/channel/composer "Hyperledger Composer 社群") 及 [Stack Overflow ![外部鏈結圖示](images/external_link.svg "外部鏈結圖示")](https://stackoverflow.com/questions/tagged/hyperledger-composer "標上 [hyperleder-composer] 的 Stack Overflow 問題")。

## 提交支援案例
{: #support-cases}

若為與「入門範本方案」或「企業方案」相關的問題，您可以在 [{{site.data.keyword.cloud_notm}} 服務入口網站 ![外部鏈結圖示](images/external_link.svg "外部鏈結圖示")](https://ibm.biz/ibmcloudsupport "IBM Cloud 服務入口網站") 中提交支援案例。

**附註：**如果您需要 Hyperledger Fabric 鏈碼、Hyperledger Composer 或應用程式方面的協助，請使用上述的社群資源，或參與 [{{site.data.keyword.blockchainfull_notm}} 服務 ![外部鏈結圖示](images/external_link.svg "外部鏈結圖示")](https://www.ibm.com/blockchain/services "使用 {{site.data.keyword.blockchainfull_notm}} 服務將區塊鏈策略轉換為業務成果")。建議僅將 Hyperledger Composer 用於展示及概念驗證。{{site.data.keyword.IBM_notm}} 不支援在正式作業中使用 Hyperledger Composer 的網路，包括 Composer CLI、JavaScript API、REST 伺服器及「Web 遊樂場」。

支援案例的回應取決於 [{{site.data.keyword.cloud_notm}} 支援方案 ![外部鏈結圖示](images/external_link.svg "外部鏈結圖示")](https://console.bluemix.net/docs/get-support/index.html#support-plans "支援方案")。

- 如果您已購買**超值支援**或**進階支援**，則可以在問題單上指定從 Sev-1 到 Sev-4 的嚴重性層次。Sev-1 是最高的嚴重性層次。具有較高嚴重性層次的支援案例表示較為緊急，因此支援團隊會更快速地回應該案例。如需相關資訊，請參閱 [{{site.data.keyword.cloud_notm}} 支援方案供應項目 ![外部鏈結圖示](images/external_link.svg "外部鏈結圖示")](https://console.bluemix.net/docs/get-support/index.html#support-plans "支援方案")。  
- 如果您未購買支援，則 {{site.data.keyword.cloud_notm}} 的「隨收隨付制」或「訂閱」帳戶會隨附免費的**基本支援**方案。在此情況下，您的支援案例將會自動登錄為 Sev-4。

請遵循下列步驟來提交支援案例。

1. 使用 {{site.data.keyword.IBM_notm}} ID 登入 [{{site.data.keyword.cloud_notm}} 服務入口網站 ![外部鏈結圖示](images/external_link.svg "外部鏈結圖示")](https://ibm.biz/ibmcloudsupport "IBM Cloud 服務入口網站")。
2. 按一下中間的**取得協助**虛擬人像，或右上角的**建立案例**按鈕。
3. 在**建立案例**表單中，至少填寫下列欄位的資訊。
  - 選擇**技術支援**作為您的案例類型。
  - 在**帳戶**欄位中，選取您用來建立區塊鏈服務實例的帳戶。
  - 在**服務地區或環境**欄位中，選取您建立服務實例所在的地區。
  - 針對**受影響的服務**欄位，選取**區塊鏈**。
  - 針對**種類**欄位，選取**服務**。
  - 在**提供組織**和**提供空間**欄位中，指定您建立區塊鏈服務實例所在的 Cloud Foundry 組織和空間。您可以從 {{site.data.keyword.cloud_notm}} 主控台找到此資訊。
  - 在**提供資源**欄位中，輸入您的區塊鏈網路 ID。若要尋找網路 ID，可以在「網路監視器」中按一下**概觀**畫面右上角的**連線設定檔**按鈕。
  - 在**案例摘要**欄位中，輸入簡短的問題摘要。
  - 在**案例說明**欄位中，盡可能詳細說明您的問題。例如，建置網路所在的資料中心，以及造成問題的步驟順序。請注意，您提供的資料愈詳細，就愈有機會可以快速解決您的問題。
4. 按一下**提交**按鈕。您會在幾分鐘內收到此案例的電子郵件通知。

若要尋找您先前提交的案例，可以按一下「{{site.data.keyword.cloud_notm}} 服務入口網站」右上角的**我的案例**。按一下案例並將其開啟，以查看其狀態或提供其他資訊。
